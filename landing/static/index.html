Готов к автоматизации?

Клонируй репозиторий, запусти мастер настройки и начни откликаться за несколько минут.

terminal
$ git clone https://github.com/nasyrov-ai/hh-auto-apply.git
$ cd hh-applicant-tool
$ pip install -e .
$ hh-applicant-tool setup # мастер настройки
$ hh-applicant-tool apply # начать отклики
