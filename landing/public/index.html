Архитектура

Три компонента, одна база. Каждый пользователь владеет своими данными.

Дашборд

Деплой на Vercel

  • Next.js 16 + React 19
  • shadcn/ui + Tailwind 4
  • Палитра команд (Cmd+K)
  • Синхронизация в реальном времени

Supabase

Свой PostgreSQL

  • Очередь команд
  • Данные откликов
  • Подписки в реальном времени
  • Row-level security

Воркер

VPS / Docker

  • Python CLI процесс
  • Опрос очереди команд
  • Запросы к API hh.ru
  • Генерация AI-писем
Дашборд→ Supabase→ Воркер→ hh.ru API

Готов автоматизировать?

Клонируй репо, запусти мастер настройки и начни откликаться за минуты.

Открыть на GitHub
terminal
$ git clone https://github.com/nasyrov-ai/hh-applicant-tool.git
$ cd hh-applicant-tool
$ pip install -e .
$ hh-applicant-tool setup
